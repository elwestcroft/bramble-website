# Bramble marketing site

Static marketing site for Bramble (macOS novel-writing software) at https://trybramble.app. No framework, no dependencies: hand-written `index.html` (single-file SPA) + `privacy.html`, with blog pages generated by `node build.mjs` from `content/<slug>.md` (frontmatter + Markdown). Run the build after any content change; generated output (`blog/`, `articles.js`, `sitemap.xml`, `robots.txt`) is committed. Pushes to `main` auto-deploy via Vercel — treat every push as production.

## Design Context

Internal design docs live at `PRODUCT.md` (strategy: audience, positioning, belief ladder, anti-references) and `DESIGN.md` (visual system: palette, typography, components, named rules), with a machine-readable sidecar at `.impeccable/design.json`. **All three are gitignored — local-only, never committed** (this is a public repo served at the domain root). Read them before any design work; if they're missing on a fresh clone, ask the owner for them.

Quick anchors (full detail in DESIGN.md):
- North Star: "The Writer's Thicket" — dark literary canvas, cream ink, sage interaction, crimson only as rare punctuation.
- Fonts are self-hosted in `/fonts/` (never re-add font CDNs).
- Real app screenshots and real quotes only — no mockups, no invented praise.
- Every animation must honor `prefers-reduced-motion`; content never gates on JS.
